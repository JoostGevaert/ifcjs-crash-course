CSS selector hierarchy

  1. id=, # CSS selectors
  2. class=, . CSS selectors
  3. HTML element CSS selectors

In the CSS style sheet file, one should go from most generic to most specific. This refers to the hierarchy listed above, but also to the hierarchy of HTML elements. This serves a dual purpose. Firstly, it keeps the code clean and therefore easier to understand and maintain. Secondly, it prevents CSS bugs.

Relational selectors

Relational selectors can be used to refer to the children of basic selectors. e.g. div > p refers to all p that are children of a <div>.
