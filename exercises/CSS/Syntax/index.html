Hello style from the <style></style> HTML element!

Cascading Style Sheets (CSS) are files that prescribe the style of HTML elements. A CSS file is linked, i.e. imported into an HTML document by adding the <link/> tag to the <head></head> of an HTML document, in which the style.css referenced.

Hello style with inline CSS!

Styling HTML using inline CSS is done by adding CSS to the style= attribute of an HTML element. Inline CSS overwrites CSS styling that is defined in the linked style.css.

CSS style hierarchy

CSS style has a hierarchy, i.e. style from the CSS file is overwritten by style in the <style></style> HTML element, which in turn is overwritten by inline CSS.

Style from:

  1. inline CSS
  2. <style></style> HTML element
  3. style.css file
